## 常见部署架构

### 标准 Web 应用部署

```
Nginx (反向代理 + SSL)
  ├── 前端静态文件 (Vue/React SPA)
  └── /api/* → 后端服务 (Spring Boot / Node.js)
                    └── 数据库 (MySQL / PostgreSQL)
                    └── 缓存 (Redis)
```

### Docker Compose 配置要点
1. 服务依赖顺序（健康检查确保启动顺序）
2. 数据持久化卷挂载
3. 环境变量配置
4. 网络隔离（内部网络和外部网络）
5. 资源限制（CPU/内存）
