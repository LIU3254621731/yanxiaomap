## 使用说明

1. **任务状态更新**：各 Agent 完成工作时更新对应任务状态
2. **通信记录**：所有重要通信和决策记录在此
3. **问题跟踪**：记录所有需要解决的问题及其状态
4. **风险登记**：记录可能影响项目进度的风险
5. **验收检查**：集成协调 Agent 在集成阶段逐项检查
