# 部署检查清单

## 概述
- **用途**: 确保部署过程顺利，系统稳定运行
- **适用阶段**: 测试环境部署、生产环境部署
- **负责人**: deployment+test Agent

## 预部署检查
### 代码检查
- [ ] 代码已通过所有测试（单元测试、集成测试）
- [ ] 代码已通过代码审查
- [ ] 代码已合并到部署分支
- [ ] 版本标签已创建

### 构建检查
- [ ] 依赖包版本已锁定
- [ ] 构建脚本已测试
- [ ] 构建产物完整
- [ ] 构建日志无错误

### 配置检查
- [ ] 环境配置文件已准备（开发、测试、生产）
- [ ] 敏感信息已移出代码（API密钥、数据库密码）
- [ ] 配置文件模板已更新
- [ ] 配置参数已验证

## 数据库部署检查
- [ ] 数据库备份已完成
- [ ] 数据库迁移脚本已测试
- [ ] 迁移脚本幂等性已验证
- [ ] 数据迁移已测试
- [ ] 回滚脚本已准备

## 服务部署检查
### 后端服务
- [ ] 服务启动脚本已准备
- [ ] 服务端口已配置
- [ ] 服务健康检查已配置
- [ ] 服务日志配置已准备
- [ ] 服务监控配置已准备

### 前端服务
- [ ] 静态资源构建完成
- [ ] CDN配置已准备
- [ ] 路由配置正确
- [ ] API代理配置正确

### 第三方服务
- [ ] 高德地图API密钥已配置
- [ ] 短信/邮件服务配置已准备
- [ ] 对象存储配置已准备

## 网络部署检查
- [ ] 域名解析已配置
- [ ] SSL证书已配置
- [ ] 防火墙规则已配置
- [ ] 负载均衡配置已准备
- [ ] CDN配置已准备

## 安全部署检查
- [ ] 访问控制列表（ACL）已配置
- [ ] 安全组规则已配置
- [ ] 入侵检测已配置
- [ ] 日志审计已配置
- [ ] 备份策略已配置

## 监控部署检查
- [ ] 应用监控已配置（APM）
- [ ] 服务器监控已配置（CPU、内存、磁盘）
- [ ] 日志收集已配置
- [ ] 告警规则已配置
- [ ] 仪表板已配置

## 部署执行检查
- [ ] 部署计划已制定
- [ ] 部署窗口已确认
- [ ] 回滚计划已制定
- [ ] 沟通计划已制定
- [ ] 应急预案已制定

## 部署后检查
### 功能检查
- [ ] 核心功能测试通过
- [ ] 用户登录注册正常
- [ ] 地图显示正常
- [ ] 数据查询正常
- [ ] 管理后台正常

### 性能检查
- [ ] 页面加载时间达标
- [ ] API响应时间达标
- [ ] 并发用户数达标
- [ ] 资源使用率正常

### 安全检查
- [ ] 漏洞扫描通过
- [ ] 渗透测试通过
- [ ] 安全头配置正确
- [ ] 敏感信息未泄露

## 检查频率
- **每次部署前**: 全量检查
- **每月**: 安全部署检查
- **每季度**: 全面部署检查
